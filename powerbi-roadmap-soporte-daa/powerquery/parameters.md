# Parámetros M requeridos

- `pSourceMode` (Text): `CSV`, `WIQL`, `OData`.
- `pCsvPath` (Text): ruta relativa del CSV para modo local.
- `pAdoUrl` (Text): URL de organización Azure DevOps.
- `pAdoCollection` (Text, opcional): colección si aplica.
- `pAdoProject` (Text): proyecto Azure DevOps.
- `pAdoTeam` (Text): team Azure DevOps.
- `pAdoQueryId` (Text): ID de query WIQL guardada.
- `pODataUrl` (Text): URL OData (uso futuro).

No guardar credenciales en parámetros ni en el repositorio.
